# HTML 정리

##  div, h1~6, p, ul, ol, li : 영역(block 요소)
## a, img, 글씨 : 영역 안에 표현되는 내용(inline 요소)
## class, title 등 태그의 속성을 Attribute 라고 한다.
##

## Emit 예
```html

html>head>title{hello}^body>h1{dasom}+p{내용}
	
	<html>
<head>
	<title>hello</title>
</head>
<body>
	<h1>dasom</h1>
	<p>내용</p>
</body>
</html>

```


# CSS 정리
```css
h2 {
border: 2px solid red;
/* 바깥쪽 여백 : margin (top left right bottom) 시계방향 */
/* margin: 100px */
/* margin-left: 30px */
/* margin-top: 50px */
margin: 50px 0 0 30px;
}
```

## 태그, id, class

## CSS의 우선순위 점수
### tag: 1점 ex. h1
### class: 10점 ex. .logo (.logo =10점 < h1.logo =11점>) 
### id: 100점 ex. #mainNavi
### (id, class, tag) 1 1 0 > 1 0 1
#### 똑같은 점수면 마지막이 우선 적용.
